# todo
Todo de tarefas, onde conseguimos adicionar novas tarefas, excluir, verificar o total e mudar o status para realizadas ou não!

Nesse projeto, utilizamos: 
  * Next
  * React
  * Styled-Components
  * Radix-ui
  * uuid
  * useEffect, useState
